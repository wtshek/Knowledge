# Some commonly used md syntax

| Element        | Syntax                               |
| -------------- | ------------------------------------ |
| Header         | \#H1, \#\#H2, \#\#\#H3               |
| Bold           | \*\*bold text\*\*                    |
| Italic         | \*italic text\*                      |
| Strike through | ~~ ~~                                |
| Highlight      | == ==                                |
| Blockquote     | \> blockquote                        |
| Code           | \`code\`                             |
| Code Block     | \``` ```                             |
| HR             | ----                                 |
| Link           | [ title ] (https://www.example.com ) |
| Image          | ! [alt text ] ( image.jpg )          |
| Emoji          | :joy: : joy :                        |

Reference: https://www.markdownguide.org/cheat-sheet/
